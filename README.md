## 📖 프로젝트 소개
**Lawmadi OS**는 기존 AI의 환각(Hallucination) 문제를 해결하기 위해 설계된 차세대 리걸테크 시스템입니다. 
대한민국 **국가법령정보센터(DRF)** 데이터를 기반으로 하며, **60인의 AI 법률 전문가(SWARM Engine)**가 협업하여 법적 근거가 확실한 답변만을 제공합니다.

이 저장소(`Lawmadi-Public`)는 Lawmadi OS의 **기술 백서(Technical Whitepaper)** 및 **아키텍처 명세**를 공개하여 기술적 투명성을 증명하기 위해 운영됩니다.

## 🏗️ 핵심 기술 (Core Architecture)
Lawmadi OS는 **Zero Trust** 보안 모델과 **SSOT** 아키텍처를 기반으로 작동합니다.

1. **Zero Hallucination:** DRF API에 없는 법령이나 판례는 절대 생성하지 않습니다.
2. **Temporal Engine:** 사건 발생 시점에 따른 '행위시법' 적용을 자동화합니다.
3. **Fail-Closed:** 데이터 무결성이 확인되지 않으면 답변을 거부하여 사용자를 보호합니다.

(자세한 기술 명세는 상단 `llms.txt` 또는 `docs/` 폴더의 백서를 참조하세요.)
